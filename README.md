# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas + JavaScript 实现的经典坦克大战网页游戏。

## 🎯 游戏特色

- 🎨 **精美的像素风格画面** - Canvas 绘制，粒子特效、屏幕震动、履带动画
- 🗺️ **3 个精心设计的关卡** - 每关有不同的地图布局和难度
- 🤖 **智能敌人 AI** - 敌人会随机移动并追踪玩家
- 💥 **丰富的战斗系统** - 连杀奖励、炮弹反弹、护盾系统
- ⚡ **道具系统** - 加速、速射、护盾、三连发
- 🔊 **简易音效** - Web Audio API 实现射击、爆炸等音效
- 📱 **移动端支持** - 触摸屏操作适配
- 🎯 **连杀系统** - 连续击杀获得额外分数奖励

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| `W A S D` / `方向键` | 移动坦克 |
| `空格` / `J` | 发射炮弹 |
| `P` | 暂停 / 继续 |
| `R` | 重新开始 |
| `Enter` | 开始游戏 |

## 🗺️ 地图元素

- 🧱 **砖墙** - 可被炮弹摧毁，可能掉落道具
- 🔩 **钢墙** - 不可摧毁，炮弹会反弹
- 🚩 **基地** - 保护目标，被击中则游戏失败

## ⚡ 道具类型

| 道具 | 效果 |
|------|------|
| ⚡ 加速 | 临时提升移动速度 |
| 🔥 速射 | 短时间内快速射击 |
| 🛡️ 护盾 | 抵挡一次伤害 |
| 💥 三连发 | 一次发射三颗子弹 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏，无需任何构建工具或服务器。

```bash
# 或者使用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- Web Audio API
- CSS3 动画

## 📝 游戏截图

游戏包含完整的侧边信息面板，显示分数、关卡、击杀数、生命值和连杀状态。
